# Project AW

A web application with responsive design and interactive features.

## Project Structure

```
aw/
├── index.html              # Main HTML entry point
├── css/
│   ├── global.css         # Global styles and variables
│   ├── layout.css         # Layout and grid styles
│   └── sections.css       # Section-specific styles
├── js/
│   ├── main.js            # Main JavaScript entry point
│   └── modules/
│       ├── animationObserver.js    # Animation observation utilities
│       ├── mouseTracker.js         # Mouse tracking functionality
│       ├── navbar.js               # Navigation bar functionality
│       └── utils.js                # Utility functions
└── assets/
    ├── fonts/             # Custom fonts
    ├── icons/             # Icon assets
    └── images/            # Image assets
```

## Features

- Responsive design
- Interactive animations with Intersection Observer
- Mouse tracking capabilities
- Dynamic navigation bar
- Modular JavaScript architecture

## Getting Started

1. Open `index.html` in your web browser
2. The application will load with all styles and scripts automatically

## File Descriptions

- **main.js** - Initializes and orchestrates all modules
- **animationObserver.js** - Handles animation triggers based on element visibility
- **mouseTracker.js** - Tracks and responds to mouse movements
- **navbar.js** - Manages navigation bar interactions and behavior
- **utils.js** - Common utility functions used across modules

## Browser Compatibility

This project uses modern web standards including:

- ES6 JavaScript
- CSS Grid and Flexbox
- Intersection Observer API
